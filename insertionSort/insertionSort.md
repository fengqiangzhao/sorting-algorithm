# 插入排序

### 原理

插入排序的原理非常简单, 类似于打牌的时候每次只考虑一张牌

### 复杂度

复杂度分析: 插入排序所需的时间取决于输入中元素的初始顺序.

- 最坏情况下 n^2 /2次比较 和 n^2 /2次交换;
- 最好情况下 n-1次比较 和 0次交换

|  平均时间复杂度  |  最好情况   |   最坏情况   | 空间复杂度 |
|     ----      |    ----    |    ----     | ---- |
|    n²    |   n  |   n²   |   1  |

### 稳定性

1. 稳定, 待插入的元素与有序序列中的某个元素相等, 则将待插入元素插入到相等元素的后面
